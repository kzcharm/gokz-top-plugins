# GOKZ Top Plugins

SourceMod plugins for integrating your GOKZ server with [GOKZ.top](https://gokz.top).

## 🔑 API Key Setup

1. Go to [gokz.top](https://gokz.top)
2. Login via Steam
3. Navigate to **Public Servers**
4. Click **API Keys** to create a new API key
5. Edit `cfg/sourcemod/gokztop/apikey.cfg` and add your key:

```sourcepawn
sm_gokztop_apikey "YOUR_API_KEY_HERE"
```
